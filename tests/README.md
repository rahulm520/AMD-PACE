# Tests
This directory contains the tests for the project.

## Unit Tests

To run all the unittests, follow the instructions below:

```
cd tests
python -m unittest [-v]
```

To run a specific test, follow the instructions below:

```
cd tests
python -m unittest tests.*test_module*.*TestClass*.*test_method*
```

> Replace `test_module`, `TestClass`, and `test_method` with the appropriate values.
